# AirdropScout

AirdropScout is an open source project designed to track and analyze potential crypto airdrops.

The project focuses on identifying early signals from emerging Web3 protocols.

## Signals tracked

testnet activity

developer ecosystem

funding rounds

community growth

## Goal

Help crypto researchers discover airdrop opportunities early.

## Structure

tracker → airdrop tracking

signals → early indicators

strategies → farming strategies

datasets → protocol datasets
